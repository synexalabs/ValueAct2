# Valuact - Actuarial Solutions Platform

A comprehensive actuarial solution platform designed for life insurance professionals to access advanced tools and capabilities across all critical areas of actuarial practice.

## Features

- **Six Professional Solution Areas**: Life Insurance Solutions, IFRS 17, Solvency II, Pricing & Products, Risk Management, Regulatory & Compliance
- **AI Assistant**: Powered by Google Gemini for professional guidance and automation
- **Advanced Calculators**: Interactive tools for complex actuarial calculations
- **Modern UI**: Built with React, Tailwind CSS, and professional design
- **Real-time Analysis**: AI-powered assistance for immediate solutions to complex problems

## Quick Start

1. **Install dependencies**:
   ```bash
   npm run install-all
   ```

2. **Set up environment**:
   - Copy `.env.example` to `.env` in the server directory
   - Add your Google Gemini API key: `GOOGLE_API_KEY=your_key_here`

3. **Run the application**:
   ```bash
   npm run dev
   ```

4. **Access the app**:
   - Frontend: http://localhost:5173
   - Backend API: http://localhost:3001

## Technology Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: Firebase Firestore
- **AI**: Google Gemini API
- **Styling**: Tailwind CSS with custom actuarial theme

## Solution Areas

### 1. Life Insurance Solutions
Core tools, product analytics, mortality tables, and actuarial principles for life insurance products.

### 2. IFRS 17 Implementation
Comprehensive tools for IFRS 17 including measurement models (GMM, PAA, VFA), CSM mechanics, and compliance reporting.

### 3. Solvency II & Capital Management
Solvency II framework tools, capital requirements, internal models, and strategic capital management for life insurers.

### 4. Pricing & Product Development
Advanced pricing techniques, product design, GLM modeling, and market analysis for life insurance products.

### 5. Risk Management & ALM
Asset-liability management, interest rate risk, longevity risk, and comprehensive risk frameworks for life insurers.

### 6. Regulatory & Compliance
Regulatory frameworks, compliance requirements, reporting standards, and emerging regulations affecting life insurers.
